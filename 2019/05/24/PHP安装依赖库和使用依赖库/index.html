一， PHP安装composer 依赖库，使用依赖库下载导出表格插件

  1. 安装依赖库

    https://www.phpcomposer.com/

    根据上面网址去下载和安装好composer

  2. 使用win+r ，输入cmd进入命令行模式，输入composer回车，看到composer及相关信息，表示安装成功；

3.然后命令行cd一波操作进入你的项目文件夹根目录下 执行 composer install 或者composer update 之后会在你项目根目录新建一个vendor文件夹 这个文件夹就是用来存放你所有下载的插件，类似于node.js的npm install 一样下载依赖

在这之前 先在你项目根目录新建一个composer.json文件
